Formatos de Imagens

Aqui nós vamos conhecer dois tipos de imagens sendo:

JPEG - Um estilo de imagem com alta quantidade de pixels ( Imagens Grandes ). É considerado também uma imagem de baixa qualidade ao dar zoom.

PNG - Um estilo de imagem com fundo transparente e de baixa quantidade de pixels ( Imagens Pequenas ). É considerado uma imagem de alta qualidade ao dar zoom.
